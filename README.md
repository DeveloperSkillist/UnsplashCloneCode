# 📝 사용
Unsplash API를 사용하고 있어, Unsplash API의 accessKey가 필요합니다.<br>
PhotosAPI.swift 파일의 fetchPhotos 메소드에 본인의 Unsplash API - accessKey를 입력하세요.<br>
https://unsplash.com/developers<br>
